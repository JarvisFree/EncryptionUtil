## Background

平时需要记录的各个账号密码等敏感信息比较多
- 直接明文记录（不安全）
- 通过现有的加解密工具（由于算法是基本是一样的 且解密时较麻烦 故不采用）
- 一般加解密工具 只要解密破解一个密文后 算法就暴露了（算法不是动态的）

基于以上痛点诞生出了此工具（EncryptionUtil）
